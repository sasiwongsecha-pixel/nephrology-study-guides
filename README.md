# Nephrology Study Guides

Interactive board-prep guides based on **Brenner & Rector's *The Kidney* (12th ed.)**.

## Files

| File | What it covers |
| --- | --- |
| **`Sodium Handling Deep Dive v2.html`** | Segmental Na⁺ handling deep dive. Five data-driven SVG transporter diagrams (PCT, LOH thin limbs, TAL, DCT, CD/CNT) with animated ion flow, lumen↔cell↔blood hover tooltips, paracellular pathways, and **6 disorder pathophysiology overlays** (Bartter, Gitelman, Liddle, FHHt/Gordon, AME, PHA-I) — each showing the broken transporter plus a defect→mechanism→labs cascade. |
| **`nephrology_v3.html`** | Broader nephrology board-prep study guide. |

## How to use

Open either HTML file directly in a modern desktop browser (Chrome, Edge, or Firefox). Everything is **self-contained** — no build step, no server, no internet needed once downloaded.

Hover any transporter box for the Brenner detail. Scroll into a diagram to see the ion-flow animation start.
