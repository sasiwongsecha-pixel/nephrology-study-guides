# Nephrology Study Guides

Interactive board-prep and fellowship-orientation guides based on **Brenner & Rector's *The Kidney* (12th ed.)** with KDIGO, Banff, and Thai Transplantation Society overlays where relevant.

## Files

| File | What it covers |
| --- | --- |
| **`Sodium Handling Deep Dive v2.html`** | Segmental Na⁺ handling deep dive. Five data-driven SVG transporter diagrams (PCT, LOH thin limbs, TAL, DCT, CD/CNT) with animated ion flow, lumen↔cell↔blood hover tooltips, paracellular pathways, and **6 disorder pathophysiology overlays** (Bartter, Gitelman, Liddle, FHHt/Gordon, AME, PHA-I) — each showing the broken transporter plus a defect→mechanism→labs cascade. |
| **`nephrology_v3.html`** | Broader nephrology board-prep study guide. |
| **`Kidney Transplant — Fellow Orientation v1.html`** | Fellowship-orientation guide for kidney transplantation, written for a Thai nephrology fellow with zero KT knowledge. **Round 1 (4 tabs):** Overview (with interactive post-tx complication timeline widget) · HLA & Crossmatch · Immunosuppression · Rejection. Each section uses a 3-layer progressive disclosure (always-visible core box → expandable detail → 🇹🇭 Thai-practice callout) plus open-ended "✓ Check yourself" Q&A. Sources: Brenner Ch 68-70 · KDIGO 2009 + 2020 · Thai Transplant Care (TTC) 1-4 · Banff 2022 (Naesens, AJT 2024). **Round 2 forthcoming:** Infections · Long-term · Recurrent disease · Comprehensive quiz. |

## How to use

Open any HTML file directly in a modern desktop browser (Chrome, Edge, or Firefox). Everything is **self-contained** — no build step, no server, no internet needed once downloaded.

- **Sodium guide:** Hover any transporter box for the Brenner detail. Scroll into a diagram to see the ion-flow animation start.
- **Transplant guide:** Click any band on the post-tx complication timeline to see its window, mechanism, and where the deep dive lives. Use the top tab strip + sticky in-tab anchor TOC to navigate. "Expand all" gives a single-scroll printable view.
